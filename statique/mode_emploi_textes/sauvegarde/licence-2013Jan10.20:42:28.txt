<h3>Quelle licence pour les contenus présents sur le site refuges.info ?</h3>

<h4>Licence</h4>
<p>
	La licence retenue pour l'ensemble des [url=##pages_points]pages des points[/url] de refuges.info est la licence "creative commons by-sa" dont une explication peut
	être trouvée à l'adresse :[url=http://creativecommons.org/licenses/by-sa/2.0/deed.fr]creative commons by-sa[/url]
</p>

<p>
L'objectif de cette licence est de garantir un maximum de liberté pour tout le monde, comme pour wikipedia.<br />
Ainsi, même si refuges.info venait à disparaître, le contenu resterait librement utilisable et copiable pour refaire un 
site similaire.<br />
Si tout le monde donne un peu sur le site, chacun peut jouir de toutes les données présentes comme il l'entend.<br />
[url=/exportations/formulaire_exportations.php]A cette adresse[/url] il est possible d'exporter TOUTE la base de refuge.info pour votre gps (format gpx) ou	vers un tableur. ( format csv )
</p>
